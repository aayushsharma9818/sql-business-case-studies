# E-commerce Funnel Analysis – SQL Case Study

This project analyzes user behavior across an e-commerce purchase funnel
to identify drop-offs and conversion opportunities.

## Dataset
RetailRocket E-commerce Dataset (Kaggle)  
https://www.kaggle.com/datasets/retailrocket/ecommerce-dataset  

*Dataset is not included due to licensing restrictions.*

## Business Objectives
- Analyze user movement across funnel stages
- Measure conversion rates
- Identify cart abandonment behavior
- Understand repeat purchase patterns

## Funnel Stages
- View
- Add to Cart
- Purchase

## SQL Concepts Used
- Aggregations
- DISTINCT counts
- Subqueries
- CTEs
- Funnel logic

## Repository Files
- `questions.md` – Business questions
- `solution.sql` – SQL analysis
- `insights.md` – Key insights & recommendations
