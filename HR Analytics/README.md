# HR Analytics – SQL Case Study

This project analyzes employee data to understand attrition patterns
and workforce behavior using SQL.

## Dataset
HR Analytics – Employee Attrition Dataset (Kaggle)  
https://www.kaggle.com/datasets/rishikeshkonapure/hr-analytics-prediction  

*Dataset is not included due to licensing restrictions.*

## Business Objectives
- Measure employee attrition rate
- Identify high-risk departments and roles
- Analyze impact of tenure, salary, and satisfaction
- Provide retention-focused recommendations

## SQL Concepts Used
- Conditional Aggregation
- GROUP BY
- CASE WHEN
- Business Segmentation

## Repository Files
- `questions.md` – Business questions
- `solution.sql` – SQL analysis
- `insights.md` – Key findings & recommendations
